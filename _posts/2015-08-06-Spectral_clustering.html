谱聚类

data(spirals)                                         ## 加载数据
sc <- specc(spirals, centers=2, kernal="rdfbot")      ## R的kernlab函数包，包含谱聚类函数
kb <- kmeans(spirals,2)                               ## R的kmeans函数

两种聚类结果的混淆矩阵如下：

final_centers = cbind(kb$centers,1)                   ##构建两种聚类的质心组合，kmeans聚类

自己实现谱聚类方法

复现上述代码，不过相似度矩阵仅用距离矩阵代替，选取特征向量也向特征值小的特征向量。最终结果如下图：

alldist <- as.matrix(dist(spirals,diag=TRUE,upper=TRUE))
sigma <- 0.05
k<- 2
#similiar <- exp(-alldist^2/(2*sigma^2))
similiar <- alldist
tmp <- diag(1/sqrt(rowSums(similiar)-1))
tmp <- tmp%*%similiar%*%tmp
diag(tmp) <- c(0)
ever <- eigen(tmp, only.values = FALSE)
vector <- ever$vectors[,299:300]
sum <- vector/rowSums(vector^2)
ab <- kmeans(sum,2)
qplot(spirals[,1],spirals[,2],color=ab$cluster)

谱聚类的思想，就是构建样本的相似矩阵，通过矩阵变换获得样本相似矩阵最主k成分，最后对特征向量进行k-means聚类，即可实现谱聚类，其实质是降维和聚类思想交叉的产物。 \( \sigma \)的选取一般和样本的均方差有关，具体规律没有确定。
